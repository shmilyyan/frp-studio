## 一、产品定位

| 项目 | 内容 |
|------|------|
| **产品名称** | FRP Studio（内部代号 Frper） |
| **目标用户** | 有内网穿透需求但不熟悉命令行的开发者、运维人员、个人用户 |
| **核心价值** | 将 frpc 客户端的配置、管理、监控全流程可视化，降低 FRP 使用门槛，提供专业的隧道管理体验 |

---

## 二、产品交付物构成

```
FRP Studio 完整交付
├── FRP Studio GUI（已实现）
│   └── Windows 桌面客户端（NSIS 安装包 / 便携版）
│       ├── 节点管理
│       ├── 隧道管理（frpc 控制）
│       ├── 实时监控
│       └── 系统设置（版本管理、Windows 服务等）
└── frps-deploy 脚本包（规划中）
    ├── install.sh        # 安装 frps + 注册系统服务
    ├── uninstall.sh      # 卸载清理
    ├── update.sh         # 版本升级
    └── frps.toml.tpl     # 配置模板
```

> frps 脚本包作为独立资源托管，GUI 内置下载入口与使用引导。

## 三、用户画像

| 类型 | 描述 | 核心诉求 |
|------|------|----------|
| 个人开发者 | 需要远程访问家庭/办公室设备 | 快速配置、开机自启、稳定重连 |

## 四、功能架构

```
FRP Studio GUI
├── 仪表盘
│   ├── 系统概览（节点数、隧道数、运行状态）
│   ├── 快捷隧道列表（最近 8 条）
│   └── 节点状态快览
├── 节点管理
│   ├── 添加 / 编辑 / 删除 FRP 服务器节点
│   ├── 连通性测试（TCP 端口探测）
│   ├── 服务端配置管理（TOML 字段编辑）
│   ├── 活动节点设置
│   └── 首次使用引导（3步教程）
├── 隧道管理（frpc 控制）
│   ├── 6 种隧道类型：TCP / UDP / HTTP / HTTPS / STCP / SUDP
│   ├── 向导式创建（3步）+ 常用场景模板
│   ├── 分组管理与筛选
│   ├── 批量启用 / 停用 / 删除
│   ├── frpc 进程启动 / 停止
│   └── TOML 配置导入 / 导出
├── 监控面板
│   ├── 实时状态（PID、运行时长、连接数）
│   ├── 流量趋势图（分钟级，最近 6 小时）
│   └── 实时日志（滚动、导出、清空）
└── 系统设置
    ├── FRP 版本管理（下载、导入、备份恢复）
    ├── 自动更新检测
    ├── 开机自启
    ├── 系统托盘
    ├── 自动重连
    ├── 代理设置（HTTP / HTTPS / SOCKS5）
    └── Windows 服务管理（独立后台运行）
```

### 5.4 新建隧道向导（3步）

**第 1 步：选择类型**
```
  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐
  │ TCP  │  │ UDP  │  │ HTTP │  │HTTPS │  │ STCP │  │ SUDP │
  │通用  │  │游戏  │  │ 网站 │  │加密  │  │安全  │  │安全  │
  └──────┘  └──────┘  └──────┘  └──────┘  └──────┘  └──────┘

  快速模板：[SSH 登录]  [Windows RDP]  [本地 Web]  [数据库]
```

**第 2 步：填写配置**
```
  节点:      [ my-server ▼      ]
  隧道名称:  [ SSH-Home          ]
  本地 IP:   [ 127.0.0.1        ]  本地端口: [ 22   ]
  远程端口:  [ 6022             ]
  分组:      [ 默认分组          ]

  ▼ 高级配置
  加密传输: [ ] / 压缩: [ ] / 带宽限制: [       ]
```

**第 3 步：确认**
```
  配置预览（TOML）:
  ┌──────────────────────────────┐
  │ [[proxies]]                  │
  │ name = "SSH-Home"            │
  │ type = "tcp"                 │
  │ localIP = "127.0.0.1"        │
  │ localPort = 22               │
  │ remotePort = 6022            │
  └──────────────────────────────┘
            [上一步]  [确认创建]
```

## 六、数据模型

### 数据库（SQLite，via sql.js）

位置：`userData/frp-studio.db`

#### nodes 表 - FRP 服务器节点

| 列名 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| name | TEXT | 节点名称 |
| host | TEXT | FRP 服务器地址 |
| port | INTEGER | FRP 服务器端口（默认 7000） |
| token | TEXT | 认证令牌（可为空） |
| created_at | INTEGER | 创建时间戳 |

#### tunnels 表 - 转发隧道配置

| 列名 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| node_id | INTEGER FK | 所属节点（级联删除） |
| name | TEXT | 隧道名称（全局唯一） |
| type | TEXT | 类型：tcp/udp/http/https/stcp/sudp |
| local_ip | TEXT | 本地 IP（默认 127.0.0.1） |
| local_port | INTEGER | 本地端口 |
| remote_port | INTEGER | 远程端口（TCP/UDP） |
| custom_domain | TEXT | 自定义域名（HTTP/HTTPS） |
| enabled | INTEGER | 启用状态（0/1） |
| group_name | TEXT | 分组名称（默认"默认分组"） |
| extra_attrs | TEXT | 扩展属性 JSON（加密、压缩、子域名等） |
| created_at | INTEGER | 创建时间戳 |

### 应用配置（config.json）

位置：`userData/config.json`

```typescript
interface AppConfig {
  trayEnabled: boolean          // 托盘模式
  activeNodeId: number | null   // 当前活动节点
  autoReconnect: boolean        // 自动重连
  reconnectDelay: number        // 重连延迟（秒）
  reconnectMaxRetries: number   // 最大重试次数（0=无限）
  autoCheckUpdate: boolean      // 自动检查更新
  updateCheckInterval: number   // 检查间隔（小时）
  proxyUrl: string              // 代理地址
  proxyEnabled: boolean         // 代理开关
}
```

### extra_attrs 扩展属性

```json
// TCP/UDP 隧道
{ "useEncryption": "true", "useCompression": "true", "bandwidthLimit": "1MB" }

// HTTP/HTTPS 隧道
{ "subdomain": "myapp", "locations": ["/api"], "httpUser": "admin", "httpPassword": "pass", "hostHeaderRewrite": "internal.host" }

// STCP/SUDP 隧道
{ "secretKey": "secret123", "allowUsers": ["user1", "user2"] }
```

## 七、技术选型

### 最终确认方案：Electron + Vue 3

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 桌面框架 | Electron 33 | 成熟稳定，跨平台 |
| UI 框架 | Vue 3 + TypeScript | 响应式，组合式 API |
| UI 组件库 | Ant Design Vue 4 | 丰富组件，深色主题支持 |
| 状态管理 | Pinia | 轻量，TypeScript 友好 |
| 路由 | Vue Router 4 | 标准方案 |
| 数据库 | sql.js（SQLite WASM） | 零依赖，纯 JS，免编译 |
| TOML 解析 | @iarna/toml | 完整 TOML 规范支持 |
| 进程管理 | Node.js child_process | frpc 子进程控制 |
| 网络检测 | Node.js net.Socket | TCP 端口连通测试 |
| 构建工具 | electron-vite + Vite 6 | 快速热重载 |
| 打包工具 | electron-builder 25 | NSIS 安装包 |
| 包管理器 | pnpm | 高效，节省磁盘空间 |

### 方案选型对比

| 方案 | 框架 | 包体 | 内存 | 开发效率 |
|------|------|------|------|----------|
| **已选** | **Electron + Vue3** | **~120MB** | **中** | **高** |
| 备选 | Tauri + Vue3 | ~10MB | 低 | 中（需 Rust） |
| 备选 | Flutter Desktop | ~30MB | 中低 | 中 |

## 十一、设计决策记录

| 序号 | 问题 | 决策结论 | 备注 |
|------|------|----------|------|
| 01 | 服务端管理方式 | frps 通过 shell 脚本交付，GUI 专注 frpc 客户端管理，两者解耦 | 兼容性更好，职责清晰 |
| 02 | 连通性测试范围 | 仅测试 frps 服务端（TCP 端口连通），不测试 Token 鉴权 | 轻量快速，避免依赖 frps API |
| 03 | 数据库方案 | sql.js（SQLite WASM）替代 better-sqlite3 | 免编译，无原生依赖，electron 兼容性更好 |
| 04 | 技术框架 | Electron + Vue3 + Ant Design Vue | 优先开发效率，生态成熟 |
| 05 | UI 主题 | 深色科技风（Dark Tech）| 适合开发者长时间使用，视觉层次清晰 |
| 06 | 隧道类型 | 支持 6 种（TCP/UDP/HTTP/HTTPS/STCP/SUDP） | 覆盖 frpc 主流代理模式 |
| 07 | 配置持久化 | 节点/隧道用 SQLite，应用配置用 config.json | 结构化数据用数据库，轻量配置用 JSON |
| 08 | Windows 服务 | 集成 winsvc，支持独立后台运行 | 满足企业用户开机无感知启动需求 |
| 09 | 版本管理 | 支持多版本备份和恢复 | 降低版本升级风险 |
| 10 | 打包方式 | NSIS 安装包（allowToChangeInstallationDirectory: true） | 用户可自定义安装目录 |
